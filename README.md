# OCaml Route Finder (Dijkstra's Algorithm)

OCamlを用いて、関西圏の鉄道網における最短経路探索プログラムを実装しました。

## プロジェクト概要
ダイクストラ法（Dijkstra's algorithm）を関数型言語で実装し、指定された2駅間の最短距離と経路を算出します。
データ構造として二分探索木（Association List形式）を採用し、効率的な駅情報の管理と更新を行っています。(Kansaiなど依存モジュールは大学の演習用ライブラリ使用)

## 特徴
- **アルゴリズム**: ダイクストラ法による単一始点最短経路問題の解決
- **データ構造**: Treeモジュールを用いた、関数型的な状態更新の実装
- **型安全性**: OCamlの強力な型システムを活かし、駅名（県名と駅名のペア）や経路リストを厳密に定義

## 使用技術
- 言語: OCaml
- アルゴリズム: ダイクストラ法

## 実行例

- 三重県内の近距離探索

$ ./dijkstra hiratacho kintetsutomida

最短距離は24.9です。最短経路は
平田町(三重) -> 三日市(三重) -> 鈴鹿市(三重) -> 柳(三重) -> 伊勢若松(三重) -> 箕田(三重) -> 長太ノ浦(三重) -> 楠(三重) -> 北楠(三重) -> 塩浜(三重) -> 海山道(三重) -> 新正(三重) -> 近鉄四日市(三重) -> 川原町(三重) -> 阿倉川(三重) -> 霞ヶ浦(三重) -> 近鉄富田(三重) -> 到着

- 存在しない駅名を指定した場合のエラーハンドリング

$ ./dijkstra hiratacho kisaragi

hiratachoまたはkisaragiは存在しません。

- 三重県（川原町）から兵庫県（出屋敷）への長距離探索

$ ./dijkstra2 mie kawaramachi hyogo deyashiki

最短距離は164.kmです。最短経路は
川原町(三重) -> 近鉄四日市(三重) -> ... (中略) ... -> 尼崎(兵庫) -> 出屋敷(兵庫) -> 到着

- 不正な県名を指定した場合（例外処理）

$ ./dijkstra2 popo kawaramachi hyogo deyashiki

Fatal error: exception Not_found
